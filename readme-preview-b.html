Setup

快速开始

前提条件：安装 Docker Desktop

1 拉取预构建镜像

最简单 — 无需克隆仓库，下载即用

# 下载 docker-compose.yml
curl -O https://raw.githubusercontent.com/aspskys/seavideo/main/docker-compose.yml

# 启动所有服务
docker compose up -d
升级注意 — 测试版版本间数据库不兼容，升级前请先清除旧数据。 
docker compose down -v
docker rmi ghcr.io/aspskys/seavideo:latest
curl -O https://raw.githubusercontent.com/aspskys/seavideo/main/docker-compose.yml
docker compose up -d
2 克隆仓库 + Docker

完全控制 — 本地构建镜像

git clone https://github.com/aspskys/seavideo.git
cd seavideo
docker compose up -d
3 本地开发模式

开发者 — 运行开发服务器

git clone https://github.com/aspskys/seavideo.git
cd seavideo && npm install

# 只启动基础设施
docker compose up mysql redis minio -d

# 运行数据库迁移 + 启动开发服务器
npx prisma db push
npm run dev
访问 http://localhost:13000（方式 1、2）或 http://localhost:3000（方式 3）开始使用。首次启动自动完成数据库初始化。
性能提示 — HTTP 下浏览器可能限制并发。安装 Caddy 启用 HTTPS 后访问 https://localhost:1443
