快速开始

前提条件：安装 Docker Desktop

方式一：拉取预构建镜像

最简单 — 无需克隆仓库

# 下载 docker-compose.yml
curl -O https://raw.githubusercontent.com/aspskys/seavideo/main/docker-compose.yml

# 启动所有服务
docker compose up -d
测试版版本间数据库不兼容。升级请先运行 docker compose down -v 清除旧数据。
方式二：克隆仓库 + Docker

完全控制

git clone https://github.com/aspskys/seavideo.git
cd seavideo
docker compose up -d
方式三：本地开发模式

开发者

git clone https://github.com/aspskys/seavideo.git
cd seavideo && npm install
docker compose up mysql redis minio -d
npx prisma db push
npm run dev
访问 http://localhost:13000（Docker）或 http://localhost:3000（开发）开始使用。
性能提示：安装 Caddy 启用 HTTPS 可解决浏览器并发限制问题。
API 配置

启动后进入设置中心配置 AI 服务的 API Key，内置配置教程。

目前仅推荐官方 API，第三方兼容格式后续持续优化。
