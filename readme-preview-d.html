// QUICK START
快速开始

前提条件：安装 Docker Desktop

拉取预构建镜像

最简单 — 无需克隆仓库，下载即用

# 下载配置并启动
curl -O https://raw.githubusercontent.com/aspskys/seavideo/main/docker-compose.yml
docker compose up -d
测试版版本间不兼容，升级需先 docker compose down -v
克隆仓库 + Docker

完全控制 — 本地构建镜像

git clone https://github.com/aspskys/seavideo.git
cd seavideo && docker compose up -d
本地开发模式

开发者 — 运行开发服务器

git clone https://github.com/aspskys/seavideo.git
cd seavideo && npm install
docker compose up mysql redis minio -d
npx prisma db push && npm run dev
访问 http://localhost:13000（Docker）或 http://localhost:3000（开发模式）开始使用。自动初始化数据库。
性能提示 — HTTP 并发受限时，安装 Caddy 启用 HTTPS 访问 https://localhost:1443
