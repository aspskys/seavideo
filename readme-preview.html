seavideo

AI 漫剧制作工作台

基于 AI 技术的短剧 / 漫画视频制作工具
支持从小说文本自动生成分镜、角色、场景，并制作成完整视频

⚡ 快速开始

前提条件：安装 Docker Desktop

最简单 — 无需克隆仓库，下载即用

# 下载 docker-compose.yml
curl -O https://raw.githubusercontent.com/aspskys/seavideo/main/docker-compose.yml

# 启动所有服务
docker compose up -d
升级注意 — 当前为测试版，版本间数据库不兼容。升级请先清除旧数据： 
docker compose down -v
docker rmi ghcr.io/aspskys/seavideo:latest
curl -O https://raw.githubusercontent.com/aspskys/seavideo/main/docker-compose.yml
docker compose up -d

启动后请清空浏览器缓存并重新登录，避免旧版本缓存导致异常。

完全控制 — 克隆仓库后构建

git clone https://github.com/aspskys/seavideo.git
cd seavideo
docker compose up -d

更新版本：

git pull
docker compose down && docker compose up -d --build

开发者模式 — 本地运行开发服务器

git clone https://github.com/aspskys/seavideo.git
cd seavideo
npm install

# 只启动基础设施
docker compose up mysql redis minio -d

# 运行数据库迁移
npx prisma db push

# 启动开发服务器
npm run dev
访问 http://localhost:13000（预构建 / Docker）或 http://localhost:3000（本地开发）开始使用！首次启动会自动完成数据库初始化，无需任何额外配置。
性能提示 — HTTP 模式下浏览器可能限制并发连接。可安装 Caddy 启用 HTTPS，然后访问 https://localhost:1443

⚙ API 配置

启动后进入设置中心配置 AI 服务的 API Key，内置配置教程。

目前仅推荐使用各服务商官方 API，第三方兼容格式（OpenAI Compatible）尚不完善，后续版本会持续优化。
